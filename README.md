# Система мониторинга вакансий

Веб-приложение для автоматического мониторинга вакансий различных специальностей, управления источниками данных, настройки правил поиска и просмотра аналитики по найденным объявлениям.

## Стек

- Frontend: `Vite + TypeScript + HTML + CSS + Chart.js`
- Backend: `Node.js + Express + TypeScript`
- Хранение данных: `SQL.js` с локальным сохранением SQLite-файла
- Тесты: `Vitest + Supertest`

## Возможности

- панель мониторинга со сводными показателями и журналом событий;
- каталог вакансий с фильтрацией, таблицей и карточкой изменений;
- управление правилами мониторинга и источниками данных;
- аналитика по динамике вакансий, компаниям и качеству источников;
- ручной запуск цикла мониторинга;
- локальное хранение данных без внешних сервисов.

## Запуск

```bash
npm install
npm run dev
```

Клиентская часть работает через Vite, сервер API запускается параллельно.

## Проверка

```bash
npm run check
```

## Production

```bash
npm run build
npm start
```

По умолчанию production-сервер доступен по адресу `http://localhost:4173/`.
